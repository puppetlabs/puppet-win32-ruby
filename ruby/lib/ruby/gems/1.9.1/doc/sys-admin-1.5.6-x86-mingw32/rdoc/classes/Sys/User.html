Constants

TEMP_DUPLICATE = 0x0100   An account for users whose primary account is in another domain.
NORMAL = 0x0200   Default account type that represents a typical user.
INTERDOMAIN_TRUST = 0x0800   A permit to trust account for a domain that trusts other domains.
WORKSTATION_TRUST = 0x1000   An account for a Windows NT/2000 workstation or server that is a member of this domain.
SERVER_TRUST = 0x2000   A computer account for a backup domain controller that is a member of this domain.

Public Class methods

new() {|self if block_given?| ...}

Creates an returns a new User object. A User object encapsulates a user account on the operating system.

Yields self if a block is provided.

Public Instance methods

Sets the account type for the account. Possible values are:

  • User::TEMP_DUPLICATE
  • User::NORMAL
  • User::INTERDOMAIN_TRUST
  • User::WORKSTATION_TRUST
  • User::SERVER_TRUST
